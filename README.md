# jogo-lebre-tartaruga
## Visão geral
Jogo da Lebre e a Tartaruga
Primeiro projeto criado em C feito na licenciatura, cujo o objetivo é conseguir recriar o antigo jogo de tabuleiro inspirado no conto "A Lebre e a Tartaruga".
Este programa deve ser capaz de premitir as seguintes fucnionalidades basicas: ao utizador jogar um nova partida contra um bot, continuar um jogo que não foi terminado, consultar as regras.
O programa deve começar com um menu com as opções disponiveis para o utilizador selecionar. Na partida deve conter todos os elementos do jogo original: tabuleiro, personagens e cartas. As regras estão escritas um ficheiro de texto chamado "Regras" e todo o jogo é criado com a assistencia da biblioteca lab.h criada e fornecida pelo professor.
## Parte tecnina
Este projeto combina conhecimentos intermedios, como estruturas de decisão, loops, menus e utilização de biblitecas, com conhecimentos mais avançados, data structures, leitura de ficheiros, manipulção de memoria, conhecimento da tabela asci ecriação de elementos visuais no terminal de output, da linguagem de C.
